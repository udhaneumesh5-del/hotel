# 🏨 Hotel Management System (PHP + MySQL)

This is a simple **Hotel Management System** built using PHP and MySQL.  
It helps to manage hotel operations like room booking, check-in, check-out, and customer management.

---

## 🚀 Features

- Add Rooms
- Manage Customers
- Room Booking System
- Check-in / Check-out System
- View Bookings
- View Available Rooms

---

## 🛠️ Technologies Used

- PHP (Backend)
- MySQL (Database)
- HTML (Frontend)
- CSS (Design)
- XAMPP Server

## ⚙️ How to Run This Project

1. Download or clone this project
2. Copy folder to:

C:\xampp\htdocs\

3. Open XAMPP and start:
- Apache
- MySQL
4. Import database:
- Open phpMyAdmin
- Create database: `hotel`
- Import: `db/hotel_management.sql`
5. Open browser:

http://localhost/hotel/


---

## 🗄️ Database Setup

- Create database name: `hotel`
- Import SQL file from:

db/hotel_management.sql
